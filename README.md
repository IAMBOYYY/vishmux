# VISHMUX 🤖

An open-source local AI agent CLI — like Claude Code or Gemini CLI, but yours.

## Quick Start

```bash
git clone https://github.com/YOURUSERNAME/vishmux
cd vishmux/cli
pip install -r requirements.txt
python setup_wizard.py
```

## Project Structure

```
vishmux/
├── cli/                  ← Main agent (runs locally)
│   ├── config.py         ← Config management
│   ├── setup_wizard.py   ← First-run setup
│   └── requirements.txt
└── README.md
```
